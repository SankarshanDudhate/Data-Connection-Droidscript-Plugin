NetChecker

NetChecker plugin is a simple plugin that checks if your device is connected internet or not.

In order to use NetChecker, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "NetChecker" );

Then you can create an instance of the plugin object when you need it like this:

 net = app.CreateNetChecker();

GetVersion() method returns the plugin version.

 net.GetVersion();

You can check if device has an active data connection using the IsConnected() method.

 isConnected = net.IsConnected();

You can get active connection type using the GetConnectionType() method. This method returns "WiFi", "Mobile Data" or "No Connection".

type = net.GetConnectionType();

You can use GetSubType() method to get the sub type of the active network.

SubType = app.GetSubType();

You can get the extra info for the active network using the GetExtras() method. This returns the SIM operator name for mobile data.

extras = net.GetExtras();

Examples:

Example - Get Version

app.LoadPlugin( "NetChecker" );

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Press Me" );
  btn.SetOnTouch( CallPlugin );
  lay.AddChild( btn );

  net = app.CreateNetChecker();

  app.AddLayout( lay );
}

function CallPlugin()
{
  app.Alert( net.GetVersion() );
}

Example - Get Type And Sub Type

app.LoadPlugin( "NetChecker" );

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Press Me" );
  btn.SetOnTouch( CallPlugin );
  lay.AddChild( btn );

  net = app.CreateNetChecker();

  app.AddLayout( lay );
}

function CallPlugin()
{
  if( net.IsConnected() === true) {
    app.Alert( net.GetType() );
    app.Alert( net.GetSubType() );
}
}
